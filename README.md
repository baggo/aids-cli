# Aids-CLI
> MaKeS eVeRy OtHeR lEtTeR cApItaLiZeD but in CLI.

## Installation

Install aids onto your system

```npm
npm install aids-cli -g
```

## Usage

#### `aids [options] [text]`

Turn the text into alternating capital and lowercase letters.

#### Example

```sh
aids aids
# AiDs

aids -o aids
# aIdS
```
